Decree JS

A fluent API for responding to keyboard input

Here's how you would respond to a single keypress. 

decree('a').perform(function() {
    console.log('"a" was pressed');
});
Go ahead and try it out by opening up your console and typing 'a' while the page is in focus. The decree method takes in a single argument, the key you are asking decree to listen for. The perform method takes in a function as a callback to be invoked once the key sequence happens. perform is always the last method called on a decree.

To make a chain of sequential key presses, use the then method. 

decree('a').then('s').then('d').then('f').perform(function() {
    console.log('"asdf" was pressed');
});
Note that the last method called here was perform.

Decree will execute the first perform method it finds along a key sequence. So if you execute the following code snippets, only the shorter decree will ever be executed. 

decree('w').then('o').perform(function() {
    console.log('I will be called!');
});
decree('w').then('o').then('o').perform(function() {
    console.log('I will never be called because the "wo" key sequence is shorter than "woo"');
}

Decree also supports key combinations instead of sequences, like pressing "shift" and "a" at the same time. After a key is specified with decree or then use the and method for the modifier key you want. 

decree('a').and('shift').perform(function() {
    console.log('"shift" and "a" were pressed at the same time.');
});
